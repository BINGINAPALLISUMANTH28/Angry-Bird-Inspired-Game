# Angry Birds Game (Pygame)

Project Description

This is a simple Angry Birds-style game developed using Python and Pygame.
The player can drag and release a bird using a slingshot to hit pigs and score points.

Features

* Slingshot shooting mechanism
* Gravity-based bird movement
* Collision detection with pigs and blocks
* Score tracking system

🛠️ Technologies Used

* Python
* Pygame

How to Run

1. Install Python
2. Install pygame:

   ```
   pip install pygame
   ```
3. Run the game:

   ```
   python angry_birds_game.py
   ```

Controls

* Mouse drag → Aim
* Mouse release → Shoot
* R → Restart level
* ESC → Exit

Output

Game window opens where you can shoot birds at pigs.
